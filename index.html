關於自己

你好，我是葉佳寶，我是一名充滿熱情的前端工程師，專注於創建優雅且動態的網頁應用程式。熟練 Vue.js 和現代前端技術，善於處理 API 串接和 Docker 打包。我注重用戶體驗和網頁動畫效果，並具備客戶協調經驗。

目前在鼎泰勝股份有限公司當前端工程師。主要工作內容是負責集團網站網站前端功能開發及子公司點餐系統開發，在此期間獨立完成網頁功能及動態效果呈現，包括專案管理、參與使用者訪談、Vue專案編寫、api串接、docker打包、Nginx架設。

在此期間因公司性質，從客戶到架設網站，都必須獨立完成，所以當面對困難時，反而能以各種角度去思考問題並且解決，或許僅有一張PDF就要能製作出該有的畫面，但程式帶給我是快樂的，特別是完成後的成就感，期待能有更加深入學習網頁技術的機會。
